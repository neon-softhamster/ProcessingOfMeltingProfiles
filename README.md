# ProcessingOfMeltingProfiles
## Required packages:

- pandas
- openpyxl
- xlrd

## Install:

```sh
$ git clone https://github.com/neon-softhamster/ProcessingOfMeltingProfiles
```
```sh
$ pip install pandas openpyxl xlrd
```
```sh
$ mkdir Raw Processed
```

Set the required parameters in the ***task.txt*** file and run ***main.py***
